# Aguedita Asalde Atelier

Sitio web de alta costura para **Aguedita Asalde Maza** — Lima, Perú.

## Tecnologías

- HTML5 semántico
- CSS puro (variables CSS, animaciones, modo oscuro/claro)
- JavaScript vanilla
- Google Fonts: Cormorant, Cormorant SC, Jost

## Despliegue

Este sitio está conectado a **Netlify** y se despliega automáticamente con cada `git push` a `main`.

## Estructura

```
/
├── index.html       # Página principal
├── images/          # Fotografías
│   ├── aguedita-1.webp
│   └── aguedita-2.webp
├── netlify.toml     # Configuración de Netlify
└── README.md
```
